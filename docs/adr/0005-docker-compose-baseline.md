# ADR 0005: Docker Compose como baseline

- `Status`: aceito
- `Contexto`: a stack precisa subir facil em uma unica maquina e ser documentavel.
- `Decisao`: usar `Docker Compose` como baseline operacional.
- `Consequencias`: instala e evolui facil; deploy mais refinado com `systemd` ou Kubernetes fica para fases futuras.
